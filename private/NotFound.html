O código 404 indica “página não existente”, isto é, a URL (endereço do site) levou você a lugar nenhum.

Code Status 404

O que causa o erro 404?

Conheça as causas mais comuns que levam ao incômodo erro 404:

Endereço digitado incorretamente. Página tirada do ar, ou seja, o endereço existia antes, mas agora sumiu. Página cuja URL foi alterada por algum motivo.

Como resolver o erro 404 not found?

A solução definitiva não existe, afinal os usuários podem continuar digitando o endereço errado. Então resta a você “tratar” o problema ao longo do tempo para minimizar as consequências.

Se você está curioso sobre como identificar o erro 404 not found no seu site, use o Google Search Console, ferramenta grátis.
